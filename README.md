# sw

A new Flutter application by suishanwen@icloud.com.

## Getting Started

flutter packages pub run build_runner build


IOS>  flutter build ios --release

打包:新建Payload文件夹，复制构建生成Runner到Payload文件夹，压缩Payload文件夹并重命名后缀为.ipa

安装:Usb连接电脑，通过 cydia Impactor，拖入ipa包并输入icloud账号完成安装

续签:越狱后通过ReProvision自动续签


Android>  flutter build apk
